# \<nickel-color-cell\>

A dynamical color cell that is customizable and parametrized.

```html
<nickel-color-cell
  min-color="[128,128,0]"
  max-color="[0,0,225]"
  nb-colors="25"
  level="3">
</nickel-color-cell>
```
